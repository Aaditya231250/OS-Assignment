# xv6 Source Code Directory

Place your xv6 source code here.

## Option 1: Download from Course
Download the xv6 tarball provided by your instructor and extract it here.

## Option 2: Clone from GitHub
```bash
git clone https://github.com/mit-pdos/xv6-public.git .
```

## Verify Installation
After adding the source code, you should see files like:
- Makefile
- syscall.c
- vm.c
- proc.c
- trap.c
- kalloc.c
- And many more...

Once the source is in place, follow the instructions in `../docs/QUICKSTART.md` to begin development.
